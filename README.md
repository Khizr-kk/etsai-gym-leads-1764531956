# GymLeads India

Auto-generated by ETS-AI 10x.

## Idea Summary
A SaaS platform for small local gyms in India to capture, track, and close leads from WhatsApp and Instagram.

## Problem Statement
Small local gyms in India lack an efficient system to manage and convert leads generated through WhatsApp and Instagram.

## Target Persona
Indian small gym owner

## MVP Features
- WhatsApp Business API integration for lead capture and communication
- Instagram DM integration for lead capture and communication
- Centralized Lead Inbox/Dashboard for all incoming inquiries
- Lead Profile Management (name, contact, source, notes, history)
- Customizable Lead Status Tracking (e.g., New, Contacted, Follow-up, Converted, Lost)
- Manual Lead Addition for offline inquiries
- Automated Follow-up Reminders based on lead status or custom schedule
- Basic Reporting: Leads by Source, Conversion Rate, Lead Pipeline Overview
- User Authentication and Authorization (single gym owner account)

## Tech Stack
Python (Django/FastAPI) for backend (good for ML integration), React/Vue.js for a responsive frontend, PostgreSQL for the database, and AWS/GCP for scalable cloud infrastructure. Utilizes WhatsApp Business API and Instagram Graph API for integrations.
